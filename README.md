## Tools

* `total-recall` - local ollama chat with RAG from your history, KV-store for your notes and chat history 

## Agents

* `file-agent` - access your projects, read, write, str\_replace, etc. for dev only
* `email-agent` - using for organizing emails
* `photo-organizer-agent` - query photos from exif tags, use YOLOv8 for object detection
* `photo-raw-agent` - convert RAF images into JPEGs using natuire language instructions (make it look like a rainy night, blah, blah...)

## Misc

* `photo-edit-models` - various flows for photo editing, mostly of experimental value (also for other research areas, e.g. depth-map from single image view, etc)
